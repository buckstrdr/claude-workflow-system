# Claude Code Workflow System

A comprehensive workflow system for Claude Code development, including commands, skills, quality gates, and automation.

## Contents

- **commands/** - Slash commands for common development tasks
- **skills/** - Reusable skill definitions
- **workflows/** - Workflow automation scripts
- **hooks/** - Git hook configurations
- **quality-gates/** - Quality gate tracking system
- **workflow_guide.md** - Comprehensive workflow documentation

## Quick Start

1. Copy relevant files to your `.ian/` directory
2. Install hooks using the provided hook scripts
3. Configure MCP servers using `mcp.json`
4. Follow the workflow guide for best practices

## Documentation

See `workflow_guide.md` for complete documentation.
